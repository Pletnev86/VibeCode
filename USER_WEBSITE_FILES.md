# Пользовательские файлы сайта

## Расположение
Ваши файлы сайта сохранены в: `src/user-website-backup/`

## Файлы
- `index.html` - главная страница сайта с формой регистрации
- `styles.css` - стили для сайта (если был создан)
- `script.js` - JavaScript для сайта (если был создан)

## Как использовать
Если хотите продолжить работу над сайтом:
1. Скопируйте файлы из `src/user-website-backup/` в отдельную папку для вашего проекта
2. Или создайте новую папку для сайта (например, `projects/my-website/`)
3. Используйте Self-Build для работы над сайтом в отдельной папке

## Важно
Системные файлы VibeCode (`main.js`, `preload.js`, `ui.js`, `index.html`) были восстановлены из бэкапа.
Ваши файлы сайта сохранены отдельно и не потеряны.
