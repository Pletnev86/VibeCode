Roadmapstart.md
Цель

Создать стартовую инфраструктуру Local Cursor X, которая способна:

воспринимать Vision и Roadmap

подготовить каркас проекта

запустить SelfDev Agent, способный программировать Cursor X по инструкции

автоматически выбрать модель и подготовить pipeline для самопрограммирования

Этап 1: Подготовка стартовой среды (Pre-Setup)

Задачи:

Создать папки проекта:

/LocalCursorX
  /src
  /ai
  /agents
  /docs


Сохранить стартовые документы:

/docs/vision.md (Visionstart.md)

/docs/roadmap.md (Roadmapstart.md)

Установить Electron и Monaco Editor:

npm init -y
npm install electron monaco-editor axios


Проверить локальный доступ к LM Studio (DeepSeek + Falcon).

Результат:
Проект готов к запуску SelfDev Agent.

Этап 2: Настройка AI Router и провайдеров

Задачи:

Реализовать /ai/router.js для маршрутизации задач:

DeepSeek → reasoning, объяснения, перевод

Falcon → генерация кода (английский)

OpenRouter / GPT → большие файлы, сложный анализ

Настроить авто-режим выбора модели:

Классификация задачи → выбор модели → optional перевод → исполнение

Добавить fallback на локальный DeepSeek при отсутствии сети.

Результат:
AI Router готов к интеграции с SelfDev Agent.

Этап 3: Разработка SelfDev Agent

Задачи:

Создать /agents/selfdev.js:

читает /docs/vision.md и /docs/roadmap.md

формирует план действий для генерации файлов проекта

генерирует минимальный каркас (main.js, preload.js, ui.js, router.js)

сохраняет файлы в /src

Подключить к UI или Node.js для тестового запуска.

Реализовать функцию разбора ответа LM Studio на отдельные файлы (parseFilesFromResult).

Результат:
Cursor X может создать стартовый каркас проекта.

Этап 4: Интеграция стартового UI

Задачи:

Реализовать интерфейс кнопки “Self-Build”:

<button id="selfbuild-btn">Self-Build</button>


Привязать к SelfDev Agent через фронтенд (window.api.chat) или Node.js вызов.

Выводить лог выполнения и список созданных файлов.

Результат:
Пользователь может инициировать стартовую генерацию каркаса проекта через UI.

Этап 5: Подготовка к автоматическому pipeline

Задачи:

Настроить pipeline SelfDev Agent:

Vision + Roadmap → классификация задачи

Выбор модели → перевод при необходимости

Генерация файлов → сохранение → лог

Подготовить базовый InterAgent Controller для маршрутизации:

selfdev → refactor → fix → patch → autocomplete → pc-control

Проверить базовую интеграцию с автотестами (Jest / pytest-lite).

Результат:
Cursor X готов воспринимать инструкции для дальнейшего самопрограммирования.

Этап 6: Проверка готовности к Auto Self-Development

Задачи:

Тестовый запуск SelfDev Agent:

Cursor X читает Vision + Roadmap

Генерирует стартовый каркас

Логирует результат

Проверка pipeline:

правильно выбирается модель

при необходимости перевод RU ↔ EN

файлы корректно сохраняются в /src

Результат:
Система готова к следующему шагу: запрос реального Vision и Roadmap для полноценного самопрограммирования.

Этап 7: Подготовка к запросу реального Vision и Roadmap

Задачи:

SelfDev Agent ожидает загрузки актуальных документов.

Cursor X формирует internal prompt:

"Загрузите реальные Vision и Roadmap. После их получения SelfDev Agent начнёт самопрограммирование проекта."


После загрузки документов: переход к генерации полноценного проекта и самопрограммирования функций.

Результат:
Cursor X полностью готов принять реальные Vision и Roadmap и начать самопрограммирование.
