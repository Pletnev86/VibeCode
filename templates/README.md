# Шаблоны и схемы запуска для VibeCode

Эта директория содержит шаблоны для различных типов проектов и инструкции по их запуску.

## Структура

- `apps/` - Шаблоны для приложений (Electron, Node.js, Python и т.д.)
- `websites/` - Шаблоны для веб-сайтов (React, Vue, статические и т.д.)
- `scripts/` - Шаблоны для скриптов и утилит

## Использование

ИИ автоматически выбирает подходящий шаблон на основе задачи пользователя.

### Примеры задач:

- "Создай веб-сайт" → использует шаблон из `websites/`
- "Создай приложение" → использует шаблон из `apps/`
- "Создай скрипт" → использует шаблон из `scripts/`

## Формат шаблонов

Каждый шаблон содержит:
- `template.json` - метаданные шаблона
- `instructions.md` - инструкции по запуску
- `files/` - файлы шаблона (опционально)
