# Инструкция по запуску Electron приложения

## Установка зависимостей

```bash
npm install
```

## Запуск приложения

```bash
npm start
```

## Сборка для продакшена

```bash
npm run build
```

## Структура проекта

- `main.js` - главный процесс Electron
- `preload.js` - скрипт предзагрузки
- `index.html` - интерфейс приложения
- `style.css` - стили

## Требования

- Node.js >= 16
- npm или yarn
