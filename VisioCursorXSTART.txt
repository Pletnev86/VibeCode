# Visionstart — стартовый Vision для Local Cursor X

## 1. Цель проекта

Создать минимальный рабочий каркас Local Cursor X, который:

- запускается на Electron и Node.js
- подключает локальные модели LM Studio (DeepSeek, Falcon)
- поддерживает внешние модели через OpenRouter и GPT API
- умеет принимать команды через чат-интерфейс
- может создавать, редактировать и сохранять кодовые файлы
- готов к расширению на самопрограммирование
- поддерживает авто-режим выбора модели ("Smart Auto Mode")
- может управлять ПК через AutoIt (клавиатура, мышь, окна, программы)
- умеет выполнять автотесты кода

---

## 2. Основные компоненты старта

### 2.1 Electron-приложение
- main.js — создает окно и подгружает preload.js
- preload.js — мост между фронтендом и Node.js
- index.html + ui.js — простой чат и textarea для задач

### 2.2 AI Router (ai/router.js)
- маршрутизирует запросы между LM Studio, Falcon и OpenRouter
- минимальная логика: выбор модели по типу задачи

### 2.3 SelfDev Agent (agents/selfdev.js)
- читает vision.md и roadmap.md
- генерирует задачи для системы
- создает кодовые файлы по задачам
- пишет логи действий

### 2.4 Execution Layer
- минимальный executor: запись файлов, запуск скриптов через Node.js
- возможность интеграции AutoIt для управления ПК
- возможность автотестирования (пустой раннер на старте)

---

## 3. Требования к минимальному старту

1. Node.js + npm + Electron
2. LM Studio локально с DeepSeek-R1 и Falcon
3. Возможность выполнять POST-запросы к LM Studio API
