Рекомендации
1. GitHub (основной репозиторий)
Создайте публичный репозиторий
Добавьте README с описанием проекта
Настройте Issues для задач
Используйте Projects для roadmap
Включите GitHub Discussions для обсуждений
2. Для командной разработки
Добавьте соавторов в репозиторий
Настройте ветки: main, develop, feature/*
Используйте Pull Requests для code review
Добавьте .github/workflows для CI/CD
3. Для пиара и прозрачности
GitHub Actions: автоматические билды и тесты
Releases: версии с описанием изменений
GitHub Pages: документация/сайт проекта
GitHub Sponsors: если планируете монетизацию
4. Дополнительно
Discord/Telegram: канал для общения
YouTube/Twitch: стримы разработки
Twitter/X: анонсы и обновления
Product Hunt: запуск при готовности MVP
