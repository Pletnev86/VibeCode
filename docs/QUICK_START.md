# Быстрый старт

## Запуск приложения

```bash
npm run dev
```

## Команды для доработки модулей через чат

### Базовые команды:

```
Доработай main.js: добавь логирование и обработку ошибок
```

```
Доработай ui.js: добавь валидацию ввода
```

```
Добавь в main.js функцию автоматического восстановления из бэкапа
```

### Полные примеры:

```
Доработай main.js: добавь все IPC обработчики (generate-project, send-chat-message, get-logs, read-file, write-file, run-command, analyze-project, enhance-modules), логирование через logger.js, инициализацию агентов через initAgents(), базу знаний через initKnowledgeBase()
```

```
Исправь проблему с кнопкой "Доработать модули": добавь обработчик события click и проверку существования window.api.enhanceModules
```

## Если приложение не запускается

1. **Восстанови из бэкапа:**
   ```bash
   npm run restore
   ```

2. **Или восстанови конкретную версию:**
   ```bash
   node restore-backup.js working-version-2025-12-06T04-24-49
   ```

3. **Создай новый бэкап:**
   ```bash
   npm run backup
   ```

## Автоматическое тестирование

```bash
npm test
```

## Полезные команды

- `npm run dev` - запуск приложения
- `npm run backup` - создать бэкап
- `npm run restore` - восстановить из последнего бэкапа
- `npm test` - запустить автоматические тесты
