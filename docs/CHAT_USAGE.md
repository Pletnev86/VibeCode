# Инструкция по использованию чата

## Что писать для доработки модулей

### Через кнопку "Доработать модули":
1. Нажми кнопку "Доработать модули"
2. Введи задачу, например:
   - "Добавь логирование в main.js"
   - "Улучши обработку ошибок в ui.js"
   - "Добавь валидацию в preload.js"

### Через чат напрямую:
Просто напиши в чат:
- "Доработай main.js: добавь все IPC обработчики"
- "Улучши ui.js: добавь обработку ошибок"
- "Добавь функцию X в файл Y"

## Автоматическое чтение файлов

AI автоматически читает файлы когда ты просишь:
- "покажи содержимое файла main.js"
- "что находится в файле ui.js"
- "прочитай файл src/main.js"
- "посмотри что в папке src"

**AI видит содержимое файлов автоматически** - не нужно копировать код вручную!

## Удаление файлов

Чтобы удалить файл, напиши:
- "удали test.js"
- "удалить файл src/testi.txt"
- "remove file.txt"

Система автоматически удалит файл из сообщения.

## Примеры запросов

### Создание файла:
"Создай файл test.js с функцией hello()"

### Редактирование файла:
"Добавь функцию goodbye() в main.js"
"Измени обработчик ошибок в ui.js"

### Просмотр файла:
"Покажи содержимое main.js"
"Что находится в файле ui.js"

### Удаление файла:
"Удали test.js"
"Удалить файл src/testi.txt"

### Доработка модулей:
**Через кнопку:**
1. Нажми кнопку "Доработать модули"
2. Введи задачу в появившемся окне

**Через чат (рекомендуется, если кнопка не работает):**
```
Доработай main.js: добавь все IPC обработчики, логирование, инициализацию агентов
```

```
Доработай ui.js: добавь обработку ошибок и валидацию
```

```
Добавь в main.js функцию автоматического восстановления из бэкапа при ошибке
```

```
Улучши chat-context.js: добавь кеширование структуры проекта
```

**Полный список команд см. в `docs/CHAT_COMMANDS.md`**
